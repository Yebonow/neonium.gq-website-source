HOW TO SETUP A SERVER
PLEASE CONFIGURE THE HOST.LUA THING (you can find it by going to the client folder, going to the content folder and going to the scripts folder and double clicking host.lua) BEFORE DOING THE REST.
OPEN THE CLIENT. OPEN A MAP FROM THE stuff FOLDER.
GO TO THE COMMAND BAR AND TYPE dofile('rbxasset://scripts/host.lua')
And Bob's your uncle.
