Welcome to my guide on using my Render078 Pack!

Open the folder called "render scripts". You will see 2 files; one called 1st script and one called 2nd script. Open the file called 2nd script. On the 5th line you should see this line of code:
print(game:service("ThumbnailGenerator"):click("PNG", 80, 80, hideSky, true))
What you need to do is, change the first number to whatever width you want the image to be, and change the second number to whatever height you want the image to be (in pixels, keep this as it is if you're not bothered about it).
